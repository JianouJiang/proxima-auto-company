Frequently Asked Questions

What HVAC scenarios are supported in beta?

Currently: Office diffuser placement and server room cooling. More scenarios (atriums, labs, cleanrooms) coming in Q2 2026 based on user requests.

Does FlowPrep work with Revit geometry exports?

Yes. Export your Revit model to STL format (File > Export > CAD Formats > STL). FlowPrep accepts STL files up to 50 MB. Tutorial video coming Week 3.

What if FlowPrep results don't match my ANSYS?

FlowPrep uses OpenFOAM, not ANSYS, so minor differences are expected (±8% typical). If you see >15% difference, email the founder with your geometry for manual debugging. FlowPrep is beta — always verify critical results with commercial tools.

How do I get support if a simulation fails?

Email [founder email] (to be added). Response within 24 hours (usually same day). For geometry issues, send your STL file for personal troubleshooting.

Can I use FlowPrep results in stamped deliverables?

FlowPrep is in beta. Recommended use: initial design exploration, then verify final results with ANSYS/commercial software before stamping. As FlowPrep matures, validation data will improve to support stamped work.

What file formats do you support?

Currently: STL only (max 50 MB). STEP and IGES coming in Q2 2026. Export from Revit, Rhino, SketchUp, or any CAD tool to STL format.

Can I customize boundary conditions?

Not in beta. FlowPrep uses pre-configured templates (ceiling diffusers, server racks). Custom BC editing coming in full release (Q3 2026). For now, edit the exported OpenFOAM case files manually if needed.
